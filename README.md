## 📖 Overview

**Mahila Shakti Unnati** is more than just an accounting app. It is a nature-inspired, high-performance financial companion for Women's Self-Help Groups (SHGs). It replaces manual paper ledgers with a secure, offline-first digital vault that speaks your language.

## ✨ Core Features

### 🌍 Language & Accessibility
- **Dual Language**: Instant toggle between **English** and **ಕನ್ನಡ (Kannada)**.
- **Intuitive Icons**: Visual cues for non-literate or semi-literate users to navigate with confidence.

### 📊 Smart Dashboard
- **Group Liquidity**: Real-time tracking of available cash vs. outstanding loans.
- **Social Impact Score**: A dynamic grading system (A+, B, etc.) based on group discipline and attendance.
- **Visual Trends**: Animated graphs showing savings growth over time.

### 🛡 Financial Integrity
- **Member Management**: Unique codes for every member.
- **WhatsApp Reporting**: One-click professional reports sent directly to members' phones.
- **Automatic Interest**: Calculates simple/compound interest for internal loans instantly.

---

## 🛠 Tech Blueprint

- **UI**: 100% Declarative Jetpack Compose.
- **Database**: Room Persistence Library with Flow for real-time UI updates.
- **Architecture**: Clean MVVM (Model-View-ViewModel).
- **Animations**: Motion layout & Smooth Tab transitions using `AnimatedContent`.

## 🚀 Installation & Setup

### 1. Prerequisites
- Android Studio **Ladybug** or newer.
- Gradle **8.5+**.

### 2. Clone & Run
```bash
git clone https://github.com/kavyashreenayak29/MahilaShaktiUnnati.git
```
Open in Android Studio, wait for the **Gradle Sync**, and hit **Run**.
