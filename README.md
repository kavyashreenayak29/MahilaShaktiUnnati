# Mahila Shakti Unnati

**Mahila Shakti Unnati** is a premium, bilingual digital ledger application designed specifically for Self-Help Groups (SHGs). It streamlines financial management, member tracking, and loan processing with a beautiful, intuitive interface.

## ✨ Key Features

- 🌍 **Bilingual Support**: Seamlessly switch between **English** and **Kannada** to ensure accessibility for all users.
- 📊 **Premium Dashboard**: Real-time insights into group liquidity, total savings, and social impact scores.
- 👥 **Member Management**: Track member details, codes, and individual financial history.
- 💰 **Savings Tracker**: Log weekly savings with integrated attendance tracking.
- 🏦 **Smart Loan Management**: Issue and track loans with visual progress bars and automatic interest calculation.
- 📱 **WhatsApp Reports**: Generate and share professional financial reports directly with members via WhatsApp.
- 🔒 **Offline First**: Powered by a local Room database, ensuring the app works perfectly even without an internet connection.

## 🎨 Visual Experience

The app features a "Nature-Inspired" premium theme using a soft color palette (Sage Greens, Earthy Browns, and Soft Whites) to provide a calm and professional financial management experience.

## 🛠 Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (Modern Declarative UI)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Database**: Room (Local SQLite persistence)
- **Navigation**: Custom Tab-based navigation with AnimatedContent transitions.

## 🚀 Getting Started

### Prerequisites
- Android Studio Ladybug or newer.
- Android SDK 24 (Android 7.0) or higher.
